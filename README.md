# VrimeClient static site

This folder is ready for GitHub Pages: upload the contents of this directory to a repository and enable Pages for the repository root.

Notes:
- GitHub Pages is static hosting only. PHP login, admin panel, payments, key activation, logs, user databases, and executables were removed from this public build.
- Public pages are available as `.html`: `index.html`, `buy.html`, `auth.html`, `dashboard.html`, `download.html`, `support.html`, `about-rustme.html`.
- For real accounts, payments, key issuing, and admin tools, use the PHP backup/server version on a PHP-capable hosting provider.
